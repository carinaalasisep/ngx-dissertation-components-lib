# Checkbox

The `CheckboxComponent` is a checkbox element with a customable label.

<Canvas of={CheckboxStories.Default} />

## Props

- `label`: The text label of the checkbox.
- `checked`: Boolean to verify if the the checkbox is checked.

## Usage

```html
<isep-lib-checkbox
  [checked]="false"
  [label]="'Checkbox Label'"
  (change)="change($event)"
></isep-lib-checkbox>
```
To use `<isep-lib-checkbox>` add the respective module to project module imports:

```typescript
// app.module.ts
import { CheckboxModule } from 'ngx-dissertation-components-lib';

@NgModule({ 
	imports: [CheckboxModule]
})
export class AppModule {}
```
