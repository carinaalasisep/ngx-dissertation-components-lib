# Select

The `SelectComponent` is a select element, with an error mode.

<Canvas of={SelectStories.Default} />

## Props

- `label`: The placeholder label of the select element.
- `options`: The options to list in the select element. This is of type SelectModel, with the properties id and name.
- `errorLabel`: Boolean that enables the error mode.

## Usage

```html
<isep-lib-select
  [options]="[{id: 1, name: 'Option 1'}, {id: 2, name: 'Option 2'}, {id: 3, name: 'Option 3'}]"
  [label]="'Select an Option'"
  [errorLabel]="false"
  (selectedItem)="selectedItem($event)"
></isep-lib-select>
```
To use `<isep-lib-select>` add the respective module to project module imports:

```typescript
// app.module.ts
import { SelectModule } from 'ngx-dissertation-components-lib';

@NgModule({ 
	imports: [SelectModule]
})
export class AppModule {}
```
