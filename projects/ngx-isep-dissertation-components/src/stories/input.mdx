# Input

The `InputComponent` is an input element that supports an error mode.

<Canvas of={InputStories.Default} />

## Props
- `width`: The width of the input element.
- `height`: The height of the input element.
- `placeholder`: The placeholder text to appear on the input before the user fills it in.
- `errorLabel`: Boolean that enables the error mode.

## Usage

```html
<isep-lib-input
  [width]="245"
  [height]="32"
  [placeholder]="'Enter text'"
  [errorLabel]="false"
  (valueChange)="valueChange($event)"
></isep-lib-input>
```
To use `<isep-lib-input>` add the respective module to project module imports:

```typescript
// app.module.ts
import { InputModule } from 'ngx-dissertation-components-lib';

@NgModule({ 
	imports: [InputModule]
})
export class AppModule {}
```
